The Basic Language of the web: HTML

Posted by Laura Jones on Monday,June21st 2027

All modern websites and web applications are built using three fundamental technologies:HTML,CSS and Javascript .These are the Languages of the web.

In this post,let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

WHAT is HTML?

HTML stands for Hyper Text Markup Language .It's markup language that web developers use to structure and describe the content of a webpage(not a programming language).

HTML consists of elements that describe different types of content: paragraphs,links,headings,images,etc.Web browsers understand HTML and render HTML code as websites.

IN HTML, each element is made up of 3 parts

  1. the opening tag
  2. the closing tag
  3. the actual element
